# Roadmap Project

A visual project roadmap built with Mermaid diagrams — Gantt charts, flowcharts, timelines, and more.

## 📄 View the Roadmap

See [ROADMAP.md](ROADMAP.md) for the full interactive roadmap with:

- **Timeline** — High-level phase overview
- **Gantt Charts** — Detailed development timeline & team workstreams
- **Flowchart** — Milestone dependency flow
- **Quadrant Chart** — Risk assessment matrix
- **Pie Chart** — Effort distribution per phase
- **Delivery Tracker** — Status table with targets and deliverables

> **Tip:** GitHub renders Mermaid diagrams natively — just open the file on GitHub to see the charts.
